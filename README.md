## Running the tasks

Open `main.py` and comment or uncomment one line to choose what runs:

```python
# Task 2 — clustering (three options, uncomment one)
assignment_pipeline.run_clustering(use_embeddings=True)        # submitted result: sentence embeddings k=8
# assignment_pipeline.run_clustering(clustering_method="kmeans")       # TF-IDF K-Means baseline
# assignment_pipeline.run_clustering(clustering_method="agglomerative") # TF-IDF Agglomerative

# Task 3 — anomaly detection
assignment_pipeline.run_anomaly_detection(use_ensemble=False)  # Isolation Forest (submitted result)
# assignment_pipeline.run_anomaly_detection(use_ensemble=True) # Ensemble of IF + LOF + kNN

# Task 1 — bag-of-words export
assignment_pipeline.run_bag_of_words_export()
```

The sentence embedding clustering requires the embedding cache to exist first. Run notebook 09 once to generate it, or run TF-IDF clustering which has no external dependency.

## Repository structure

```
data/
  raw/
    articles.csv                  — source dataset (2,164 documents)
  processed/
    normalized_text_views.csv     — cached normalised text (both views)
    clustering_tfidf_matrix.npz   — cached TF-IDF matrix for clustering
    anomaly_lsa_matrix.npy        — cached LSA + structural feature matrix
    sentence_embeddings.npy       — cached sentence embeddings (generated by notebook 09)
  results/
    clusters.csv                  — Task 2 submission: cluster label per document
    anomalies.csv                 — Task 3 submission: 50 anomaly document IDs
    bag_of_words_matrix.csv       — Task 1 submission: bag-of-words matrix
    notebook_*.csv                — intermediate results exported by notebooks

src/
  core/
    assignment_pipeline.py        — main orchestrator: runs preprocessing, clustering, anomaly detection
    data_io.py                    — load/save helpers for CSV and matrix files
    paths.py                      — all file paths in one place
  preprocessing/
    text_normalization.py         — two-view text normaliser (clustering view + anomaly view)
    vectorizer.py                 — TF-IDF, bag-of-words, and LSA vectorisation
    structural_features.py        — 17 hand-crafted structural features for anomaly detection
  clustering/
    kmeans_clustering.py          — K-Means with silhouette-based k selection; embedding clustering
    agglomerative_clustering.py   — Ward-linkage agglomerative clustering with SVD projection
  anomaly_detection/
    isolation_forest_detection.py — Isolation Forest detector and anomaly output builder
    lof_detection.py              — Local Outlier Factor detector
    knn_detection.py              — k-nearest neighbours distance detector
    ensemble_detection.py         — rank-average ensemble of all three detectors
  exploration/
    data_exploration.py           — corpus statistics and anomaly candidate helpers used in notebooks

notebooks/
  01_task1_advanced_exploration_and_normalization.ipynb
  02_task1_normalization_walkthrough.ipynb
  03_task2_clustering_methods_demo.ipynb
  04_task2_cluster_interpretation_and_submission.ipynb
  05_task3_anomaly_methods_and_top50.ipynb
  06_task3_lsa_anomaly_search.ipynb
  07_task3_anomaly_vectorization_comparison.ipynb
  08_task3_structural_features_analysis.ipynb
  09_task2_sentence_embedding_clustering.ipynb  — generates sentence_embeddings.npy

docs/
  report.pdf                      — submitted report
  assignment.pdf                  — original assignment description

tests/                            — unit tests (22 tests, all passing)
main.py                           — entry point
pyproject.toml                    — dependencies and tool configuration
```
